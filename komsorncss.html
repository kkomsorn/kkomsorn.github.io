CSS

CSS แบ่งได้เป็น 3 ส่วนหลักๆ ได้แก่

1.Selector

เป็นส่วนที่เอาไว้เลือกว่าจะดำเนินการกับ Element ใดๆในไฟล์ HTML

2.Property

เป็นส่วนที่ใส่ความต้องการของเราว่าจะทำอะไรกับ Element ที่ Selector กำหนดเอาไว้

3.Value

เป็นส่วนกำหนดค่าของ Property ว่าต้องการค่าใดๆเท่าไหร่บ้าง

วิธีที่เราจะนำ CSS มาใช้กับ Html มี 3 วิธี

1.External stylesheet

แยกไฟล์ CSS ออกไปเป็นอีกไฟล์หนึ่ง แล้วลิงค์กลับมายัง Html

2.Internal sylesheet

เพิ่ม Tag Style เข้าไปใน Tag Head (อยู่ในไฟล์เดียวกับ Html)

3.Inline style

คือการเขียน Style เข้าไปที่ Tag ของ Element นั้นๆโดยตรง (แบบนี้ไม่แนะนำให้ทำ)
